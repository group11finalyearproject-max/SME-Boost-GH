# SME Boost GH - Codex Instructions

## Project Overview
SME Boost GH is a mobile app for Ghanaian entrepreneurs and small business owners.

Core features:
- Business Plan Drafting
- Marketing Content Generation
- Professional Email Composition

## Goal
Improve UI/UX of the existing app to be:
- Clean
- Modern
- Smooth
- Trustworthy
- Easy for non-technical users

## Target Users
- Ghanaian SMEs
- Low to medium tech knowledge
- Need guided workflows and simplicity

## Tech Stack
- React Native (Expo)
- Existing structure must be preserved
- Backend already working — do not break API logic

## Folder Structure
- app/ → screens
- components/ → reusable UI
- backend/ → API
- navigation → routing
- constants/theme → styles

## Skills
- UI/UX Pro Max located at `.codex/skills/ui-ux-pro-max`

## Rules
- ALWAYS use UI/UX Pro Max for UI decisions
- DO NOT rebuild app from scratch
- Improve incrementally
- Preserve business logic
- Extract reusable components when needed

## Design Goals
- Strong visual hierarchy
- Consistent spacing
- Clean typography
- Clear CTAs
- Smooth user flow
- Mobile-first layout
- Good empty/loading/error states

## UX Principles
- Reduce cognitive load
- Use guided flows (step-by-step)
- Keep forms simple
- Show clear feedback
- Prioritize readability

## What “Done” Means
- UI is consistent across screens
- Navigation works properly
- Code is clean and reusable
- UX feels simple and intuitive
